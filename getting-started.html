Getting Started with Unity

Learn how to install Unity and create your first project.

1
Download Unity Hub

Visit Unity.com and download Unity Hub. Unity Hub helps manage Unity versions and your projects efficiently.

2
Create Your First Project

Open Unity Hub â†’ Click New Project. Choose a 2D or 3D template, name your project, and select where to save it.

3
Explore the Unity UI
  • Scene View: Build your game world and manipulate objects.
  • Game View: Preview how your game looks when played.
  • Hierarchy: Lists all objects in the current scene.
  • Inspector: Shows properties and settings of the selected object.
  • Project Window: Stores all assets, scripts, and prefabs for your project.
4
Save Your Project

Remember to save your work regularly with Ctrl + S (Windows) or Cmd + S (Mac).

The Unity Editor provides the tools to create, preview, and manage your game projects.
