Unity Basic Tools

Learn how to create objects, write scripts, and animate in Unity.

1
Creating Objects

Go to GameObject > 3D Object > Cube. Use the Move, Rotate, and Scale tools from the toolbar to shape it as desired.

2
Writing Scripts

Right-click in the Project Window > Create > C# Script. Name it PlayerMovement.cs and attach it to an object by dragging it onto the object in the Hierarchy.

3
Creating Animations

Open the Animation Window. Select your object, click Create Animation, and record simple changes such as movement, rotation, scaling, or color adjustments.

Use Unity's Animation window to create simple animations like movement, scaling, to your objects.

Use Unity's hierarchy, inspector, and project windows to manage objects, scripts, and assets effectively.
